# natsumica.github.io

Developer website for Natsumica apps.

This repository is intended to publish:

- `https://natsumica.github.io/`
- `https://natsumica.github.io/app-ads.txt`

## app-ads.txt

Create `app-ads.txt` at the repository root using the exact line shown in AdMob.

Typical format:

```txt
google.com, pub-0000000000000000, DIRECT, f08c47fec0942fa0
```

Replace `pub-0000000000000000` with the publisher ID shown in AdMob.
